# Reo Glossary — Anamata Kāhui

Approved kupu (words) and kīanga (phrases) for use in platform UI,
translations, and external communications. Translators should prefer
entries from this list. New entries require kaitiaki rōpū review.

**Status:** Living document. Add entries via PR with rationale and
example usage.

## Conventions

1. **Macrons are mandatory.** Tāonga not taonga. Tūhoe not Tuhoe.
   CI does not yet enforce this; reviewers must.
2. **Capitalisation:** "Māori" capitalised as a proper noun.
   "iwi" lowercased as common noun. "Ngāti X" always capitalised.
3. **Plurals:** "waiata" is its own plural. "iwi" is its own plural.
   No English-style -s on te reo words.
4. **First mention:** spell out in full ("Te Reo Māori"); subsequent
   mentions can shorten ("reo Māori").
5. **Translations:** if a kupu has no English equivalent, leave it
   untranslated with a brief gloss on first use.
